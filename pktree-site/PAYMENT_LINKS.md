# Payment Direct Links

This document contains the direct URLs that can be used to open the site with a specific payment method pre-selected. These links are useful for sharing directly with people.

Base URL: `https://peykc.github.io/pktree/`

## Direct Payment Links

Append `?pay=<method>` to the base URL to automatically open that payment method's modal.

*   **Cash App:** `https://peykc.github.io/pktree/?pay=cashapp`
*   **Venmo:** `https://peykc.github.io/pktree/?pay=venmo`
*   **Zelle:** `https://peykc.github.io/pktree/?pay=zelle`
*   **Monero:** `https://peykc.github.io/pktree/?pay=monero`
*   **PayPal:** `https://peykc.github.io/pktree/?pay=paypal`
*   **Bitcoin:** `https://peykc.github.io/pktree/?pay=bitcoin`
*   **Ethereum:** `https://peykc.github.io/pktree/?pay=ethereum`

## Additional Options

For crypto methods, you can also append `amount` and `note` parameters:

*   Example (Monero with amount): `https://peykc.github.io/pktree/?pay=monero&amount=0.5`
*   Example (Bitcoin with note): `https://peykc.github.io/pktree/?pay=bitcoin&note=Donation`
